# Mannie Travels

A personal trip planner web app.

## Deploy to Vercel

1. Install Node.js from https://nodejs.org
2. Install Vercel CLI: `npm install -g vercel`
3. Extract this zip and open the folder in terminal
4. Run: `npm install`
5. Run: `vercel`
6. Follow the prompts - your app will be live in minutes!

## Add Your Logo
Place your logo file as `public/logo.png` before deploying.
It will appear in all screen headers automatically.

## Local Development
Run `npm run dev` to preview locally at http://localhost:5173

## Notes
- Photo uploads and document attachments work fully on Vercel
- Print and email itinerary work fully on Vercel
- The app works as a PWA - users can Add to Home Screen
